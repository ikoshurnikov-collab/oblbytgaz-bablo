Сборка и Деплой на GitHub Pages

🚀 Как запустить систему на GitHub Pages:

  1. Создайте новый репозиторий на GitHub (например, oblbytgaz-crm).
  2. Скопируйте файлы index.html и app.js (коды предоставлены ниже) в этот репозиторий.
  3. В начале файла app.js замените пустые строки в объекте firebaseConfig на ваши реальные ключи базы данных Firebase Firestore (их можно взять в консоли Firebase в настройках вашего веб-проекта).
  4. Зайдите в настройки репозитория: Settings > Pages, выберите ветку main и папку / (root), затем нажмите Save.
  5. Готово! Через пару минут ваша защищенная CRM-система ведения должников будет доступна по адресу https://ваше-имя.github.io/oblbytgaz-crm/.
